## Application Function
My Application is on Phone, Firebase Auth is used for authentication and database for storing phone battery logs. 
Services/API/Capacitor are Integrated.
HttpClient is used for fetching the data from the json api.
Tabs is used and a home for Angular Routing.
Multiple Standalone Components and Multiple Ionic UI Components are used.
GEOLOCATOR FOR capacitor requirment is used.
AuthService and FirestoreService used to authentication and database crud.

## Running the Application
- Clone the repository
- cd phone-status-app
- npm install
- ionic serve
